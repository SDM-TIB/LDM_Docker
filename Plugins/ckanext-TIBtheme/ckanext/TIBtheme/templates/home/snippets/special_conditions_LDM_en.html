4.1 Anyone can use the LDM service to explore LDM datasets without registration. The metadata describing the LDM datasets can be reused freely for any purpose users see fit, in accordance with the CC BY-SA License (5.2 of these Special Conditions LDM). The LDM datasets can be reused following the regulations imposed by the license of the dataset, which is specified by the registered user who created the dataset in LDM.

4.2 LDM provides functionality that requires registration. Adding and editing, including deleting, existing datasets and their metadata is reserved for Registered Users. Metadata about the LDM datasets is created and edited in accordance with the CC0 License (5.1 of these Special Conditions LDM). Registered Users obtain LDM user profiles, can link their profile to ORCiD; the profiles include provenance information.

4.3 LDM provides programmatic access to the content (API). Anyone can use the LDM API to access existing content. Creating and editing, including deleting, existing content via the LDM API is reserved for Registered Users.

4.4 LDM enables DOI-based publication of LDM datasets. In doing so, metadata including creator names and, if known, ORCiD IDs are shared with DataCite. If a Registered User publishes an LDM dataset in this form, she or he agrees to the sharing of metadata with DataCite. The sharing of such data is based on user consent given as part of the service of DataCite e. V. and ORCiD. Registered Users need to connect their LDM profile with ORCiD and need to explicitly activate DOI-based publication of the LDM datasets.
