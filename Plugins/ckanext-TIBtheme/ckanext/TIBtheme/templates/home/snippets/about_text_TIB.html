{{ "Über den Leibniz-Datenmanager" }}

Der Leibniz Data Manager (LDM) ist eine quelloffene und kostenlose webbasierte Anwendung für das Forschungsdatenmanagement (RDM). Der LDM bietet verschiedene Distributionen an, um den spezifischen Anforderungen verschiedener Kunden gerecht zu werden, z.B. Instituten und Forschungsgruppen, die RDM in verschiedenen wissenschaftlichen Disziplinen betreiben. Die LDM-Distributionen werden von der TIB und L3S entwickelt, gepflegt und kuratiert.

Der Prototyp bietet derzeit die folgenden Funktionen zur Visualisierung von Forschungsdaten:

  • Unterstützung von Datensammlungen und Publikationen mit unterschiedlichen Formaten
  • Unterschiedliche Sichten auf denselben Datensatz (2D- und 3D-Unterstützung)
  • Visualisierung von Auto-CAD-Dateien
  • Jupyter Notes zur Demonstration von Live-Code
  • RDF Beschreibung von Datensammlungen

Die dateispezifischen Viewer wurden mit Hilfe von CKAN-Plugins (Comprehensive Knowledge Archive Network) implementiert, um bestehende Viewer für die in der CKAN-Instanz enthaltenen Datensätze darzustellen.

TDie RDM-bezogenen Konzepte, die LDM-Distributionen und die Deployment-Methoden werden im Projekt-Wiki auf GitHub erläutert.

Neben dem Leibniz Data Manager bietet die TIB weitere Dienste rund um das Forschungsdatenmanagement für Institutionen und Wissenschaftler an. Dazu gehören ein DOI-Service, Unterstützung bei der Auswahl der richtigen Repositorien, bei der Veröffentlichung und Archivierung von Forschungsdaten sowie Schulungs- und Beratungsangebote. Eine vollständige Leistungsbeschreibung finden Sie auf unserer Homepage: https://www.tib.eu/de/publizieren-archivieren/forschungsdaten

Weitere Informationen über das LDM-Projekt finden Sie auf der TIB-Webseite und den zuletzt aktualisierten Code im GitHub-Repository.

Bitte kontaktieren Sie uns unter service.ldm@tib.eu, wenn Sie Kommentare oder Fragen haben.

{{ "About the Leibniz Data Manager" }}

The Leibniz Data Manager (LDM) is an open source and free web-based application for Research Data Management (RDM). LDM delivers different distributions to best fit the specific requirements of various customers, e.g., institutes and research groups performing RDM in various scientific disciplines. The LDM distributions are designed, maintained, and curated by  TIB  and  L3S.

The prototype currently offers the following functions for the visualization of research data:

  • Supports data collections and publications  with different formats
  • Different views on the same data set (2D and 3D support)
  • Visualization of Auto CAD files
  • Jupyter Notes for demonstrating live code
  • RDF Description of data collections

The file specific viewers were implemented using CKAN (Comprehensive Knowledge Archive Network) plugins to render existing viewers for the datasets included in the CKAN instance.

The RDM arelated concepts, the LDM distributions, and the deployment methods are explained in the projects wiki on GitHub.

Besides the Leibniz Data Manager, TIB provides further services around research data management for institutions and researchers. This includes a DOI service, support in choosing the right repositories, and in publishing and archiving research data, as well as training and advisory services. Please refer to our homepage for a full service description: https://www.tib.eu/en/publishing-archiving/research-data/

More info about the LDM project can be found on the TIB webpage and the last updated code in the GitHub repository.

Please, contact us at service.ldm@tib.eu if you have comments or questions.
