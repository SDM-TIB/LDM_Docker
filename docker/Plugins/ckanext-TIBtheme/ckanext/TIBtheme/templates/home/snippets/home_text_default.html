Der Leibniz Data Manager (LDM) wurde entwickelt, um den Aspekt der besseren Wiederverwendbarkeit von Forschungsdaten für Forschungsvorhaben zu fördern. Der Prototyp unterstützt die Verwaltung und den Zugang zu heterogenen Forschungsdatenpublikationen und hilft Forschern bei der Auswahl relevanter Datensätze für ihre jeweiligen Disziplinen.

Der Prototyp bietet derzeit folgende Funktionen zur Visualisierung von Forschungsdaten:
· Unterstützung von Datensammlungen und Publikationen mit unterschiedlichen Formaten.
· Unterschiedliche Sichten auf denselben Datensatz (2D- und 3D-Unterstützung).
· Visualisierung von Auto-CAD-Dateien.
· Jupyter Notes zur Demonstration von Live-Code.
· RDF-Beschreibung von Datensammlungen.

The Leibniz Data Manager (LDM) is an open source and free web-based application for Research Data Management (RDM).

{%- if instance_name -%}

This is an instance of the tool installed for: {{instance_name}}.

{%- endif -%}

LDM was developed to support the aspect of better re-usability of research data for scientific projects. The prototype fosters the management and access to heterogeneous research data publications and assists researchers in the selection of relevant data sets for their respective disciplines.

The prototype currently offers the following functions for the visualization of research data:
· Supports data collections and publications with different formats.
· Different views on the same data set (2D and 3D support).
· Visualization of Auto CAD files.
· Jupyter Notes for demonstrating live code.
· RDF Description of data collections.
