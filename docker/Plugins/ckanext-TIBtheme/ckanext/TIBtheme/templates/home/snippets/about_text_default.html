{{ "Über den Leibniz-Datenmanager" }}

Der Leibniz Data Manager (LDM) ist eine quelloffene und kostenlose webbasierte Anwendung für das Forschungsdatenmanagement (FDM). Der LDM bietet verschiedene Distributionen an, um den spezifischen Anforderungen verschiedener Kunden gerecht zu werden, z.B. Instituten und Forschungsgruppen, die RDM in verschiedenen wissenschaftlichen Disziplinen betreiben.

Der Prototyp bietet derzeit die folgenden Funktionen zur Visualisierung von Forschungsdaten:

  • Unterstützung von Datensammlungen und Publikationen mit unterschiedlichen Formaten
  • Unterschiedliche Sichten auf denselben Datensatz (2D- und 3D-Unterstützung)
  • Visualisierung von Auto-CAD-Dateien
  • Jupyter Notes zur Demonstration von Live-Code
  • RDF Beschreibung von Datensammlungen

Die dateispezifischen Viewer wurden mit Hilfe von CKAN-Plugins (Comprehensive Knowledge Archive Network) implementiert, um bestehende Viewer für die in der CKAN-Instanz enthaltenen Datensätze darzustellen.

{{ "About the Leibniz Data Manager" }}

The Leibniz Data Manager (LDM) is an open source and free web-based application for Research Data Management (RDM). LDM delivers different distributions to best fit the specific requirements of various customers, e.g., institutes and research groups performing RDM in various scientific disciplines.

The prototype currently offers the following functions for the visualization of research data:

  • Supports data collections and publications  with different formats
  • Different views on the same data set (2D and 3D support)
  • Visualization of Auto CAD files
  • Jupyter Notes for demonstrating live code
  • RDF Description of data collections

The file specific viewers were implemented using CKAN (Comprehensive Knowledge Archive Network) plugins to render existing viewers for the datasets included in the CKAN instance.
