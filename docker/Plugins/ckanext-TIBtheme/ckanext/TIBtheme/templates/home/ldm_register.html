    Rechte und Rollen der verschiedenen Benutzer

    LDM-Besucher

    Jeder Besucher der Website kann auf der Website navigieren, die öffentlichen Datensätze durchsuchen und die verfügbaren öffentlichen Ressourcen ohne Einschränkungen ansehen und/oder herunterladen.

    Benutzer, Organisationen und Autorisierung

    LDM-Benutzer können Benutzerkonten registrieren und sich anmelden. Die Anmeldung ist nicht erforderlich, um Daten zu suchen und zu finden, aber für alle Veröffentlichungsfunktionen: Datensätze können von Benutzern mit den entsprechenden Berechtigungen erstellt, bearbeitet usw. werden. In LDM werden alle Forschungsdaten innerhalb von Organisationen und Gruppen verwaltet. Eine Organisation ist eine Institution oder Einrichtung, der der/die Forscher angehört/angehören.

    Registrierung auf der Plattform

    Wenn eine Organisation oder Gruppe zu LDM hinzugefügt werden soll, muss sich der Verantwortliche oder Vertreter an den Systemadministrator von LDM wenden, um die Organisation oder Gruppe im System anzulegen und einen Administrator-Benutzer für sie einzurichten. Dazu senden Sie bitte eine E-Mail an service.ldm@tib.eu.

    Nach Erhalt der Anfrage werden die Systemadministratoren von LDM die Organisation und die Gruppen erstellen und ihnen einen "Administrator"-Benutzer zuweisen. Ein neu registrierter Benutzer kann keine Datensätze erstellen, bis ihm eine Rolle zugewiesen wird.

    LDM erlaubt es Administratoren, Mitglieder (Benutzer) zu Organisationen/Gruppen mit drei Arten von Rollen hinzuzufügen:

    Für Organisationen:

    • Mitglied - kann die privaten Datensätze der Organisation sehen
    • Editor - kann Datensätze bearbeiten und veröffentlichen
    • Admin - kann Rollen für Organisationsmitglieder hinzufügen, entfernen und ändern

    Für Gruppen:

    • Mitglied - kann Datensätze von Gruppen hinzufügen/entfernen
    • Admin - kann die Gruppeninformationen bearbeiten und die Gruppenmitglieder verwalten.

    Um Mitglieder zu Organisationen/Gruppen hinzufügen zu können, muss der neue Benutzer den Administrator fragen, ob er seine E-Mail und seine persönlichen Daten mitbringen darf.

    Bitte kontaktieren Sie uns unter service.ldm@tib.eu, wenn Sie Kommentare oder Fragen haben.

    Rights and roles of different users

    LDM visitors

    Any visitor of the website is able to navigate the site, perform searches over the public datasets, preview and/or download the available public resources without restrictions.

    Users, organizations and authorization

    LDM users can register user accounts and log in. Login is not needed to search for and find data, but is needed for all publishing functions: datasets can be created, edited, etc by users with the appropriate permissions. In LDM all research data is managed within Organizations and Groups. An Organization is an institution or entity where the researcher/s is/are affiliated.

    Registering to the platform

    In case an organization or group wants to be added to LDM, the responsible or representative must contact LDM’s sysadmin asking for the creation of the organization or group in the system and creation of an administrator user to it. For this please send an email to service.ldm@tib.eu.

    After recieving the solitude LDM’s System Administrators will create the Organization and Groups and assign an "Aministrator" user to them. A new registered user is not able to create datasets until a role is assigned.

    LDM allows administrator users to add members (users) to organizations/groups with three kind of roles:

    For organizations:

    • Member – can see the organization’s private datasets
    • Editor – can edit and publish datasets
    • Admin – can add, remove and change roles for organization members

    For groups:

    • Member – can add/remove datasets from groups
    • Admin – can edit group information, as well as manage group’s members.

    To be able to add members to organizations/groups the new user must ask the administrator user her/his addition bringing her/his email and personal data.

    Please, contact us at service.ldm@tib.eu if you have comments or questions.
